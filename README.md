# typescript-numberguess-cli

Number Guess CLI using Node.js, TypeScript, and Inquirer

## Preview

<img src="./preview.gif" />

## Setup

```bash
# Clone repo
$ git clone https://github.com/SaadFarhanIdress/typescript-numberguess-cli

# Navigate to project
$ cd typescript-calculator-cli

# Install deps
$ npm i

# Run file
$ npm start
```
